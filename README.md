## A) Progetto di Laravel 7:
 ### Server del Database:
   * **Tipo di Server**: MySQL.
   * **Versione del server**: 8.0.18 - MySQL Community Server - GPL.
   * **Porta**: 3308.
   * **Versione PHP**: 7.3.12.
    
 ### Il Processo di Installazione:
